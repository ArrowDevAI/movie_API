movie_API Endpoints

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None A JSON Object holding data about the movies
Return Data about a single movie title /movies/[Name] GET None A JSON Object holding data about a single title
Return Data about a Genre /movies/genres/[Name] GET None A JSON Object holding data about a single genre
Return Data about a Director by name /movies/directors/[Name] GET None A JSON object holding data about a single director
Allow new users to register /users POST {
"Username": "string",
"Password": "string",
"Email": "string",
"Birthday": "string"
} 		A text response of the added username only + "added"
Allow Users to Update their user info /users/[userName] PUT {
"Username": type: "string",
"Password": type: "string",
"Email": type: "string",
"Birthday": type: "string"
} 		A text message with username " updated" if the information is successfully updated and an error message if user not found
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST {
MovieID in URL to be 24 digit UUID
}		 A JSON formatted response of the user with new MovieID in the FavoriteMovies array
Allow users to remove a movie to their list of favorites /users/:Username/movies/:MovieID DELETE JSON to be defined A JSON formatted response of the user showing the title removed from the FavoriteMovies array
Allow Existing users to de-register /users/: DELETE None A text message that a user been removed
